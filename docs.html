Developer Documentation

SDK Version v1.0.2

1 SDK Integration

To display ads, place the following script in the <head> section of your site.

<script type="module" src="https://adsero.vercel.app/sdk.js" data-app-id="YOUR_APP_ID"></script>

2 Call for advertising

Use the AdseroSDK class to display ads. The SDK must be initialized after the page has loaded.

// Reklamani chaqirish funksiyasi async function startAd() { if (window.AdseroSDK) { const ad = new window.AdseroSDK(); await ad.showInterstitial(); } else { console.error("Adsero SDK is not loaded yet."); } }
Note: Since the SDK is a module, it attaches itself to the window object. Always check if(window.AdseroSDK) before calling.
