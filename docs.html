Developer Documentation

SDK Version v1.0.2

1 SDK Integration

To start showing ads, first include the SDK script in your website's <head> or at the end of <body>. Replace YOUR_APP_ID with your project's unique ID.

<script type="module" src="https://adsero.vercel.app/sdk.js" data-app-id="YOUR_APP_ID"></script>

2 Showing Interstitial Ads

To trigger an ad, you need to initialize the AdseroSDK class and call the showInterstitial() method. You can set a custom countdown time (default is 15 seconds).

// Initialize SDK const adsero = new window.AdseroSDK(); // Show ad with 15 seconds timer adsero.showInterstitial(15);
Note: The ad will only show if there are active campaigns with enough budget in the system.

Full Example

<button onclick="showMyAd()">Watch Ad to Earn</button> <script> function showMyAd() { if(window.AdseroSDK) { const ad = new AdseroSDK(); ad.showInterstitial(10); // 10 second timer } else { console.error("SDK not loaded yet"); } } </script>
